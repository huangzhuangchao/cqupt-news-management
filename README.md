### 🌈 这是一个node.js后端+vue3前端的重邮新闻管理发布系统 全栈项目
#### 本系统分为admin端、web端以及server端
### 🛠️ 技术栈
#### Vue3 + Vue-Router + Pinia + Axios + Element-Plus + Express + MongoDB
### 🚀 项目启动
**1、Web 端**

```
// 页面启动
cd web
npm install
npm run dev

// http://localhost:5173/
```

**2、Admin 端**
```
// 页面启动
cd admin
npm install
npm run dev

// http://localhost:5174/
```

**3、Server 端**
```
// 服务启动
cd server
npm install
npm run start
```
